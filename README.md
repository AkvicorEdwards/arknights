# arknights

明日方舟/Arknights 循环刷图脚本

循环刷 1-7 和 活动图 的时候每次过关和开始都要手点太累了，所以还是交给脚本干吧。

adb连接手机（模拟器也可以，但是貌似要改命令）需要先选中想要循环刷的关卡，然后运行脚本

- [x] 根据理智判断是否继续
- [x] 可自动使用药剂
- [x] 可碎石（需指定数量）
- [x] 可在循环的任意阶段运行脚本
- [x] 计算关卡通过时间，单次循环时间

```sh
# 不使用理智药剂，不碎石
python3 main.py
# 使用理智药剂，不碎石
python3 main.py 1
# 使用理智药剂，碎n颗石头石（n为大于等于0的整数）
python3 main.py 1 1 n
```
